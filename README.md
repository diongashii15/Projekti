# Cloud Resume Hub



Aplikacion modern per menaxhimin e CV-ve ne Cloud, i ndertuar me arkitekturen React dhe Supabase.

---

### Teknologjite e Perdorura

| Sektori | Teknologjia |
| :--- | :--- |
| **Frontend** | React 18, Vite |
| **Styling** | Tailwind CSS |
| **Database** | Supabase (Postgres) |
| **Hosting** | Vercel |

---

Instalimi Lokalisht

Bash
